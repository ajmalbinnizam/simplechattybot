Write a program that calculates the factorial of the input number using while loop.

Factorial of the number N is the result of the multiplication of all positive positive integers less than or equal to N. For example, the factorial of 3 is the product of 1, 2 and 3.
.

The input format:

The number N in range from 1 to 100.

The output format:

The factorial N!.


Sample Input:
3
Sample Output:
6


Memory limit: 256 MB
Time limit: 15 seconds
